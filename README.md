# Tauben (Ninas Work)

Eigenstaendiges Next.js-Projekt fuer Ninas Work.

## Starten

1. Abhaengigkeiten installieren:
   npm install
2. Umgebungsvariable setzen (siehe .env.example):
   DATABASE_URL=...
3. Dev-Server starten:
   npm run dev
4. Browser:
   http://localhost:3000

Die App startet jetzt direkt auf `/` und zeigt Ninas Work dort an.
